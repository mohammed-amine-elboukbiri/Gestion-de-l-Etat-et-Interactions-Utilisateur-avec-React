#  React Hooks — useState & useEffect

> Ce repository vous a permis de comprendre la gestion de l'état local, les interactions utilisateur, les effets secondaires et le flux unidirectionnel des données en React.

---

## 📋 Objectifs

- ✅ Comprendre le rôle de `useState` pour gérer l'état local
- ✅ Mettre en place des interactions utilisateur (clic, saisie, soumission)
- ✅ Découvrir `useEffect` pour gérer des effets secondaires
- ✅ Respecter le flux unidirectionnel des données

---
